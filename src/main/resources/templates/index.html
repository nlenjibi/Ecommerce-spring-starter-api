Public & Home

GET /
Main landing page (rendered HTML)
GET /api/home
API information, version, features & useful endpoints

Authentication & Users

GET /v1/users/{id}
Get user details by ID (admin or self)
POST /v1/users
Register / Create new user

Products

GET /api/v1/products
List all products (advanced filtering, pagination, sorting)
GET /api/v1/products/{id} or /api/v1/products/{slug}
Get single product (by ID or SEO-friendly slug)
GET /api/v1/products/featured
Get featured / promoted products
GET /api/v1/products/category/{categoryId}
Products by category

Categories

GET /api/v1/categories
List all categories
GET /api/v1/categories/hierarchy
Full nested category tree
GET /api/v1/categories/slug/{slug}
Get category by URL-friendly slug

Cart Management

GET /carts/{cartId}
View cart contents
POST /carts/{cartId}/items
Add product to cart
DELETE /carts/{cartId}/items/{productId}
Remove specific item from cart

Orders — Most Important

GET /orders/my-orders
List current user's orders
GET /orders/admin/all?page=0&size=20&sortBy=orderDate&direction=DESC
Get ALL orders (Admin only) — paginated
POST /orders
Create new order (checkout)
PUT /orders/{id}/cancel
Cancel own order
PUT /orders/admin/{id}/ship
Mark order as shipped (Admin)

Wishlist

GET /api/v1/wishlist
Get current user wishlist
POST /api/v1/wishlist
Add product to wishlist
